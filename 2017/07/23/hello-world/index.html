In the academic theme, other subdirectories are available for pages with particular content and content-specific attributes.

Updating YAML metadata

You can, of course, update the YAML frontmatter directly, but for categories and tags you can use blogdown through the Addin menu to scan previously used categories and tags. To get this functionality select Update Metadata from the menu.
